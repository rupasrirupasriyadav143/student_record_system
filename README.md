# Student Record System

A console-based mini project developed using C language.

## Features
- Add student details
- Display student details
- Search a student using roll number
- Update student marks
- Delete a student record
- Exit the program

## Technologies Used
- C Language

## How to Run
1. Download the `student system record.c` file.
2. Compile it using a C compiler.
3. Run the program.
4. Choose an option from the menu.

## Note
This project currently stores one student record at a time.
